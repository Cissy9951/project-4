Line Cooking

Using Grasshopper in Rhino.

1. Start from base ingredients: Point, Polygon, Curve, Line, Grid.

2. Mix match secondary ingredients with base ingredients in different orders. Base and secondary ingredients may be reused in one script: Offset Curve, Rotate, Rotate Axis, Construct Point, Divide Curve, Closest Point, Explode, Deconstruct, Interpolate, Jitter/Shift List.

3. Control the amount (quantity and size) of each ingredient using: Series, Number Slider, Graph Mapper, Remap Numbers, Bounds, Construct/Deconstruct Domain.

4. Explore until ideal result reached. Ideal result is: a drawing made of lines (closed lines/shapes, or open lines) that looks like food.

5. May not display two separate (unconnected) script at the same time to make one drawing. Each drawing must be one continuous script to mimic the continual sequential nature of cooking a dish with real food.

6. Finish the dish by adding color using Custom Preview (Gradient or Color Swatch).
